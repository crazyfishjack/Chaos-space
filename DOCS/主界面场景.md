# 主界面场景 (MainScene)

## 功能概述

主界面场景是玩家登录后进入的核心界面，采用**纯白背景简约高级风格**设计，整体布局分为五大部分。

**新增特性**：支持手机圆角适配和顶部安全距离，界面不会顶到屏幕边缘。

## 界面布局

```
┌─────────────────────────────┐
│                             │
│    ↑ 顶部安全距离 (24px)     │  ← 适配刘海屏/圆角
│                             │
├─────────────────────────────┤
│ 昵称                        │  ← Header (56px)
├─────────────────────────────┤
│                             │
│          hello1             │  ← Content1 (1/3)
│                             │
├─────────────────────────────┤
│                             │
│          hello2             │  ← Content2 (1/3)
│                             │
├─────────────────────────────┤
│                             │
│          hello3             │  ← Content3 (1/3)
│                             │
├─────────────────────────────┤
│ 角色 │ 背包 │ 混沌空间 │ 历史 │ 设置 │  ← Footer (64px)
└─────────────────────────────┘
```

## 安全距离配置

### 顶部安全距离

通过 `safeArea` 配置对象设置顶部安全距离：

```javascript
this.safeArea = {
  top: 24,      // 顶部安全距离（状态栏+圆角适配）
  bottom: 0,    // 底部安全距离
  left: 0,      // 左侧安全距离
  right: 0      // 右侧安全距离
};
```

**设计说明**：
- 顶部安全距离固定为 **24px**，用于适配刘海屏和圆角屏幕
- 界面所有元素从 `y = 24` 开始绘制
- 中间三块区域高度计算时扣除顶部安全距离

### 适配效果

- **刘海屏**: 顶部 24px 留白避免昵称被刘海遮挡
- **圆角屏**: 内容不会紧贴屏幕圆角边缘
- **普通屏幕**: 顶部留白使界面更美观

## 五大部分说明

### 1. 顶部昵称栏 (Header)
- **高度**: 56px
- **背景**: 纯白色 `#ffffff`
- **位置**: `y = 24` (顶部安全距离之后)
- **内容**: 显示玩家昵称，默认显示"昵称"
- **文字样式**: 16px, font-weight: 500, 颜色 `#1a1a1a`
- **左边距**: 20px
- **分割方式**: 底部阴影渐变分割

### 2. 中间第一块 (Content1)

- **背景**: 纯白色 `#ffffff`
- **内容**: 显示 "hello1"
- **文字样式**: 32px, font-weight: 300, 颜色 `#2c3e50`, 居中显示
- **分割方式**: 底部阴影渐变分割

### 3. 中间第二块 (Content2)
- **高度**: 剩余高度的 1/3
- **背景**: 微灰色 `#fafafa` (用于视觉层次区分)
- **内容**: 显示 "hello2"
- **文字样式**: 32px, font-weight: 300, 颜色 `#2c3e50`, 居中显示
- **分割方式**: 底部阴影渐变分割

### 4. 中间第三块 (Content3)
- **高度**: 剩余高度的 1/3
- **背景**: 纯白色 `#ffffff`
- **内容**: 显示 "hello3"
- **文字样式**: 32px, font-weight: 300, 颜色 `#2c3e50`, 居中显示
- **分割方式**: 底部阴影渐变分割

### 5. 底部功能按钮栏 (Footer)
- **高度**: 64px
- **背景**: 纯白色 `#ffffff`
- **按钮**: 五个等宽按钮
  - 角色
  - 背包
  - 混沌空间

  - 直角设计（无圆角）

  - 颜色 `#1a1a1a`
  - 按钮之间用 1px 阴影线分隔
- **分割方式**: 顶部阴影渐变分割

## 核心方法

### 构造函数
```javascript
constructor(ctx, width, height, sceneManager)
```
**功能**: 初始化场景，配置安全距离和布局参数

**参数**:
- `ctx` - Canvas 2D 渲染上下文
- `width` - 画布宽度
- `height` - 画布高度
- `sceneManager` - 场景管理器实例

**初始化逻辑**:
1. 设置 `safeArea` 安全距离配置（顶部 24px）
2. 计算中间三块区域高度（扣除顶部安全距离）
3. 初始化底部五个按钮的位置和尺寸

### onEnter
```javascript
async onEnter(data = {})
```
**功能**: 场景进入时调用，检查登录状态并加载用户数据

### loadUserData
```javascript
async loadUserData()
```
**功能**: 从后端 API 获取用户信息和角色数据

### render
```javascript
render()
```
**功能**: 渲染主界面，按顺序绘制五大部分

### drawBackgroundWithRoundCorners
```javascript
drawBackgroundWithRoundCorners()
```
**功能**: 绘制带圆角的白色背景

**实现细节**:
- 使用 16px 圆角半径
- 在安全边距内绘制圆角矩形
- 调用 `clearOutsideRoundCorners()` 清除外部区域

### clearOutsideRoundCorners
```javascript
clearOutsideRoundCorners()
```
**功能**: 清除圆角矩形外部的区域，确保内容不溢出

**实现方式**: 使用 `destination-out` 混合模式清除四个角的外部区域

### 绘制方法

| 方法名 | 功能说明 |
|--------|----------|
| `drawHeader()` | 绘制顶部昵称栏（从 `y = 24` 开始） |
| `drawContent1()` | 绘制中间第一块内容区（考虑顶部安全距离） |
| `drawContent2()` | 绘制中间第二块内容区（考虑顶部安全距离） |
| `drawContent3()` | 绘制中间第三块内容区（考虑顶部安全距离） |
| `drawFooter()` | 绘制底部功能按钮栏 |
| `drawFooterButton(btn, index)` | 绘制单个底部按钮 |
| `drawShadowDivider(x, y, width)` | 绘制底部阴影分割线 |
| `drawShadowDividerTop(x, y, width)` | 绘制顶部阴影分割线 |

### 事件处理

#### onTouch
```javascript
onTouch(e)
```
**功能**: 处理触摸事件，检测底部按钮点击

**参数**:
- `e` - 触摸事件对象

**点击区域计算**: 考虑安全边距，只响应安全区域内的触摸

#### handleButtonPress
```javascript
handleButtonPress(btn)
```
**功能**: 处理按钮按下效果，提供视觉反馈（300ms高亮后恢复）

**参数**:
- `btn` - 被点击的按钮对象

## 设计特点

1. **顶部安全距离**: 固定 24px 顶部留白，适配刘海屏和圆角屏幕
2. **纯白背景**: 整体采用 `#ffffff` 纯白背景，简约干净
3. **阴影分割**: 模块之间使用柔和的阴影渐变分割，替代传统的细线
4. **字体高级**: 使用细体 (font-weight: 300/400/500) 营造高级感
5. **直角设计**: 按钮采用直角设计，现代简约
6. **无图片素材**: 纯代码绘制，不依赖外部图片资源
7. **响应式布局**: 中间三块区域根据屏幕高度自动计算

## 适配说明

### 刘海屏适配
- 顶部 24px 安全距离，避免昵称被刘海遮挡
- 所有内容区域从 `y = 24` 开始绘制

### 圆角屏适配
- 顶部留白使内容不会紧贴屏幕圆角边缘
- 界面整体向下偏移，视觉更舒适

### 普通屏幕
- 顶部留白使界面更美观，不会顶到屏幕边缘

## 注意事项

- 底部按钮目前仅提供视觉反馈，无实际功能
- 昵称显示目前为占位符，实际功能待后端接口实现
- 点击按钮时会在控制台输出日志，便于开发调试
- 顶部安全距离固定为 24px，可根据需要调整
